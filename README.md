🛍️ Myntra Clone
A high-fidelity frontend clone of the Myntra homepage, built strictly with HTML5 and CSS3 to master modern UI/UX layout techniques.
🚀 Overview
This project is a pixel-perfect replica of one of India's leading e-commerce platforms. The primary goal was to challenge my styling skills by recreating a complex, production-grade landing page without using any external CSS frameworks like Bootstrap or Tailwind.
✨ Key Features
Pure CSS Mastery: Developed entirely with vanilla HTML5 and CSS3.
Modern Layouts: Utilized CSS Flexbox and Grid to build a structured and scalable interface.
Interactive UI: Implemented smooth hover effects, transitions, and polished typography to mimic the real Myntra user experience.
Component-Based Design: Replicated essential e-commerce sections including:
Sticky Navigation Bar with dropdown-style layout.
Responsive Hero Banners and Promotional Sliders.
Dynamic Product Display Cards with pricing and branding.
🛠️ Tech Stack
HTML5: For semantic and accessible web structure.
CSS3: For custom styling, animations, and responsive positioning.
